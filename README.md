## 🛠️ Agentic Instagram
It's just a chatbot that performs actions on Instagram based on the user's intent. For now, it has basic features to show how the bot integrates with Instagram:
- sending messages (just type, for example, “ask user123 how they're doing”),
- follow a user (“follow the user with ID 13142425”),
- like a post (“like the post with ID 111241242”)
